Типы в Python

Тип - это набор допустимых значений и операций над ними. Фактически типы отвечают на вопрос: Каким способом я могу общаться с питоном? Так вот только через эти самые типы!

Основные типы
  • int - тип целочисленных значений (12, 42, -11, 108, 0)
  • float - тип чисел с плавающей запятой (дробных)(12.0, -506.1, 67.62, 11.121)
  • bool - логический тип (True, False)
  • str - строковой тип ('Hello!', 'How are you?', 'What is your name?\n')(символ '\n' - это escape-последовательность (специальный символ, начинающийся с нижнего подчёркивания), которая переносит курсор на новую строку при выводе в консоль)

Создание/использование переменных

Шаблон создание переменной: имя_переменной = значение

Например:
a = 12
b = 'Hello'
c = True
d = 2.48f

!!! ИМЕНА ПЕРЕМЕННЫХ НЕ ДОЛЖНЫ ПОВТОРЯТЬСЯ!

!!! Хорошим тоном является:

  1. Давать переменным осмысленные имена, которые отражают суть того, что в них лежит
  2. Называть переменные в стиле snake_case (каждое следующее слово называется с маленькой буквы и отделяется от предыдущего '_' (нижним подчёркиванием))
Например:
apples_count = 12
money_count = 800
can_buy_product = True
name = 'Vika'

!!! Важно не злоупотреблять длинными названиями! На ЕГЭ важна не красота и читаемость кода, а его работоспособность!

Обращение к переменным происходит через использование их имени. При обращении к переменной на её место просто ставиться значение, которой в ней хранилось.

Например:
x = 12
print(x)

Заменится на:

print(12)

Операторы

Операторы - это специальные сиволы, которые выполняют определённые действия со своим операндами, заменяя себя на другую строчку.

Операнды - объекты, с которыми работают операторы.

Виды операторов

Операторы делят на группы:

  • Унарные: У них всего один операнд (not)
  • Бинарные: У них два операнда (=, +, *, /, +=, ==, >, >=, and, or и т.д.)

Нифига не понятно?) Ну например:

12 + 3

В данном примере знак '+' - это оператор, а числа '12' и '3' - это операнды (потому что именно с ними оператор и работает).

И когда я писал про то, что операторы заменяют себя на другую строчку, в данном примере произойдёт следующая замена:

12 + 3 -> 15

То есть себя же операторы заменяют каким-то результатом.

Пример:
x = 17
y = 2 + x
print(y)

1) Подставляем вместо переменной x его значение:

y = 2 + 17
print(y)

2) Заменяем оператор(+) и операнды(2 и 17) на результат операции:

y = 19
print(y)

3) Подставляем вместо переменной y его значение:

print(19)

Какие существуют операторы (что было -> во что превратилось):

+
1 + 5 -> 6
'Hello, ' + 'World!' -> 'Hello, World!'
-
12 - 3 -> 9
*
3 * 4 -> 12
'Hi' * 4 -> 'HiHiHiHi'
/
4 / 2 -> 2.0
15 / 4 -> 3.75
//
4 // 2 -> 2
15 // 4 -> 3
%
4 % 2 -> 0
15 % 4 -> 3
7 % 8 -> 7
**
2 ** 3 -> 8
>
2 > 3 -> False
3 > 3 -> False
4 > 3 -> True
>=
2 >= 3 -> False
3 >= 3 -> True
4 >= 3 -> True
<
2 < 3 -> True
3 < 3 -> False
4 < 3 -> False
<=
2 <= 3 -> True
3 <= 3 -> True
4 <= 3 -> False
==
4 == 4 -> True
4 == 3 -> False
!=
4 != 4 -> False
4 != 3 -> True
not
not True -> False
not False -> True
and
True and True -> True
False and True -> False
True and False -> False
False and False -> False
or
True or True -> True
False or True -> True
True or False -> True
False or False -> False

Это важно:

x = 12
x + 1
print(x)

Вывод: 12

Но:

x = 12
x += 1
print(x)

Вывод: 13

Таким образом x + 1 просто превращается в результат суммы, но не перезаписывает значение в переменной x в отличие от +=

Соответственно для каждого арифметического оператора существуют свои версии подобной записи:

+=, -=, *=, /=, //=, %=, **=

Условные операторы

if условие:
    инструкции...
Правила
  1. Если условие True, тогда инструкции внутри if-а выполняются, иначе - не выполняются
  2. Инструкции считаются внутри if-а, если для них используется выравнивание равное 4-м пробелам (табуляция)
Пример:
money = 200
product_cost = 150
can_buy = money >= product_cost
if can_buy:
    print('Удалось купить продукт!')

1) Подставляем вместо переменных money и product_cost значения:

can_buy = 200 >= 150
if can_buy:
    print('Удалось купить продукт!')

2) Заменяем оператор(>=) и операнды(200 и 150) на результат операции:

can_buy = True
if can_buy:
    print('Удалось купить продукт!')

3) Подставляем вместо переменной can_buy значение:

if True:
    print('Удалось купить продукт!')

Вспоминаем правило 1. Поскольку внутри условия if-а стоит True, значит инструкция внутри него(print('Удалось купить продукт!')) будет выполнена

Вывод программы: Удалось купить продукт!

Обратный пример:
money = 80
product_cost = 150
can_buy = money >= product_cost
if can_buy:
    print('Удалось купить продукт!')

1) Подставляем вместо переменных money и product_cost значения:

can_buy = 80 >= 150
if can_buy:
    print('Удалось купить продукт!')

2) Заменяем оператор(>=) и операнды(80 и 150) на результат операции:

can_buy = False
if can_buy:
    print('Удалось купить продукт!')

3) Подставляем вместо переменной can_buy значение:

if False:
    print('Удалось купить продукт!')

Вспоминаем правило 1. Поскольку внутри условия if-а стоит False, значит инструкция внутри него(print('Удалось купить продукт!')) не будет выполнена.

Программа ничего не выведет в консоль

elif и else:

Пример:
x = 5
if x > 0:
    print('Число положительное')
if x < 0:
    print('Число отрицательное')
if x == 0:
    print('Число равно нулю')

Проблематика: в данном случае первая проверка пройдёт, но программу это не остановит от проверки оставшихся двух условий, что совершенно излишне.

Данную проблему можно исправить, если оставшиеся два слова заменить на elif:

x = 5
if x > 0:
    print('Число положительное')
elif x < 0:
    print('Число отрицательное')
elif x == 0:
    print('Число равно нулю')

теперь если первая или вторая проверка пройдут, то оставшиеся проверяться не будут, что улучшит производительность кода.

Более того последнюю проверку можно ещё сильнее упростить:

x = 5
if x > 0:
    print('Число положительное')
elif x < 0:
    print('Число отрицательное')
else:
    print('Число равно нулю')

ведь если x не больше и не меньше нуля, значит x равен нулю однозначно иных вариков быть не может.

в таком случае удобно юзать else

Циклы

Циклы позволяют выполнять определённые строчки кода по нескольку раз.

Виды циклов

1. Через range:

for x in range(1, 6, 1):
    print(x)
    print('Всё клёво')

Данный цикл превратиться в:

x = 1
print(x)
print('Всё клёво')

x = 2
print(x)
print('Всё клёво')

print('Всё клёво')

x = 4
print(x)
print('Всё клёво')

x = 5
print(x)
print('Всё клёво')

Пояснения: for x in range(1, 6, 1) -> для переменной x в диапазоне от 1 до 6 не включительно с шагом 1 выполни инструкции, написанные внутри цикла(print(x) и print('Всё клёво')).

Как бы спортсмену дают коробку названную x с изначально одной маслиной. В процессе пробежке ему встречается print(x), которая смотрит ему в коробку и выводит в консоль сколько у спортсмена маслин, и print('Всё клёво'), которая просто выводит 'Всё клёво'. До тех пор, пока маслин у нас в коробке x не станет 5 или больше, мы будем возвращаться к началу цикла, нам будут добавлять одну маслину, а мы будем проходить его занаво.

Пример с шагом в 2:
for x in range(0, 10, 2):
    print(x)
    print('Всё клёво')

Данный цикл превратиться в:

x = 0
print(x)
print('Всё клёво')

x = 2
print(x)
print('Всё клёво')

x = 4

print('Всё клёво')

x = 6
print(x)
print('Всё клёво')

x = 8
print(x)
print('Всё клёво')

После каждой пробежки добавляют по 2 маслины в коробку.

Интересный факт:

range(3, 10, 1) -> range(3, 10)
range(0, 10) -> range(10)

Полезные встроенные функции Python

print() -> выводит сообщение в консоль

input() -> приостанавливает выполнение кода, пока пользователь не введёт значение с клавиатуры и не нажмёт Enter и превращается в введённое пользователем сообщение типа str

int() -> конвертирует значение в тип int

str() -> конвертирует значение в тип str

Примеры

1.

print('Hello, World!')

Вывод: Hello, World!

2.

x = input()
y = input('Введите y: ')
print(x, y)

1) Заменяем функции input() каким-нибудь введённым с клавиатуры значением. Пусть это будет, например, 'Привет' и 'Как дела?':

x = 'Привет'
y = 'Как дела?'
print(x, y)

2) Подставляем вместо переменных x и y значения:

print('Привет', 'Как дела?')

Вывод программы:

// тут происходит ввод пользователем Привет
Введите y: // тут происходит ввод пользователем Как дела?
Привет Как дела?

3.

x = int(input())
print(x + 3)

1) Заменяем функцию input() каким-нибудь введённым с клавиатуры значением. Пусть это будет, например, '7':

x = int('7')
print(x + 3)

2) Функция int() конвертирует значение в тип int. В данном случае ей это удаётся потому она успешно превращается в:

x = 7
print(x + 3)

2) Подставляем вместо переменной x значение:

print(7 + 3)

3) Заменяем оператор(+) и операнды(7 и 3) на результат операции:

print(10)

Вывод программы:

// тут происходит ввод пользователем 7
10

!!! Если в данном примере ввести с клавиатуры не число, то функция int() не сможет его сконвертировать из типа str в тип int и программа выдаст ошибку
